# 技术栈

## 框架与运行时

- **平台**: 微信小程序
- **基础库版本**: 2.32.3
- **渲染引擎**: Skyline + glass-easel 组件框架

## 开发语言

- **TypeScript**: 主要开发语言，严格模式启用
- **SCSS**: 样式预处理器
- **WXML**: 微信小程序模板语言

## 编译配置

- TypeScript 目标: ES2020
- 模块系统: CommonJS
- 启用严格空值检查、隐式 any 检查等严格类型检查

## 依赖

- `miniprogram-api-typings`: 微信小程序 API 类型定义

## 编辑器配置

- 缩进: 2 空格
- 使用空格而非 Tab
